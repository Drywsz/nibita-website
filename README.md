# NIBita Website

Este repositório contém o código-fonte do site da Nova Igreja Batista de Itacoatiara (NIBita), um projeto desenvolvido para a comunidade local. O site apresenta informações sobre a igreja, seus ministérios, eventos e outras atividades.

## 🚀 Tecnologias Utilizadas

*   **HTML5:** Estrutura semântica do conteúdo.
*   **CSS3:** Estilização e responsividade do layout.

## ✨ Funcionalidades

*   Páginas informativas sobre a igreja.
*   Seções dedicadas a ministérios e eventos.
*   Design responsivo para diversas telas.

## ⚙️ Como Rodar Localmente

1.  Clone este repositório:
    ```bash
    git clone https://github.com/Drywsz/nibita-website.git
    ```
2.  Navegue até o diretório do projeto:
    ```bash
    cd nibita-website
    ```
3.  Abra o arquivo `index.html` no seu navegador preferido.
